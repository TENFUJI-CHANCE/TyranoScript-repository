### <a id="playbgm"></a> **[playbgm] 播放背景音乐**

该标签用于播放背景音乐。背景音乐文件请放在data/bgm目录下。

推荐使用mp3格式，也支持ogg、wav和m4a格式。注意，ogg格式某些浏览器并不支持。不支持的情况下，建议同时放m4a和ogg，以便其他浏览器正常播放音频文件。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定要播放的背景音乐文件||〇|
|loop|设定背景音乐的播放是否循环|true|
|sprite_time|设定背景音乐播放的起始时间和结束时间，以毫秒为单位设定。例如，“6000-10000”。在v515及以上版本中，可以使用“mm:ss”格式。例如，“00:06-00:10”|
|volume|设定播放音量，数值在0到100之间|
|html5|设定是否使用HTML5的音频，通常使用网络音频接口（false）即可。该属性设定的必要性极低|false|
|pause|设定是否要暂停播放。暂停播放后，请用[resumebgm]标签继续播放|false|
|seek|设定播放的起始时间。例如，该属性被设置为4500，则会从4.5秒处开始播放|
|restart|该属性用于判断一首音乐在播放途中，使用[playbgm]标签再次播放，是否从头开始，如果设置为“true”则会从头开始播放|

代码结构及示例：
```
[playbgm storage="music.mp3" loop="true" sprite_time="6000-10000" volume="80" html5="false" pause="false" seek="0" restart="false"]
```

### <a id="xchgbgm"></a> **[xchgbgm] 以淡入淡出模式将当前背景音乐替换成另一个**

【不推荐使用】该标签用于两首背景音乐之间的切换，当前正在播放的背景音乐会以淡出模式结束播放，新背景音乐以淡入模式开始播放。这样使得背景音乐的切换更加自然和丝滑。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定接下来要播放的背景音乐||〇|
|loop|设定背景音乐的播放是否循环|true|
|time|设定淡入淡出的时间，以毫秒为单位|2000|

代码结构及示例：
```
[xchgbgm storage="music.mp3" loop="true" time="2000"]
```
  
### <a id="playse"></a> **[playse] 播放效果音**

该标签用于播放效果音。背景音乐文件请放在data/sound目录下。

推荐使用mp3格式，也支持ogg、wav和m4a格式。注意，ogg格式某些浏览器并不支持。不支持的情况下，建议同时放m4a和ogg，以便其他浏览器正常播放音频文件。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定要播放的效果音文件||〇|
|buf|设定要播放音效的声道。这个声道理论上来说是无限的，无需担心设置，在设定这个属性的时候，声道也就设定了|0|
|loop|设定背景音乐的播放是否循环|false|
|sprite_time|设定背景音乐播放的起始时间和结束时间，以毫秒为单位设定。例如，“6000-10000”。在v515及以上版本中，可以使用“mm:ss”格式。例如，“00:06-00:10”|
|clear|设定是否要停止所有声道的效果音的播放|false|
|volume|设定播放音量，数值在0到100之间|
|html5|设定是否使用HTML5的音频，通常使用网络音频接口（false）即可。该属性设定的必要性极低|false|

代码结构及示例：
```
[playse storage="se.ogg" buf="0" loop="false" sprite_time="3000-5000" clear="false" volume="100" html5="false"]
```

### <a id="wbgm"></a> **[wbgm] 等待背景音乐播放完毕**

该标签将等待背景音乐播放完毕。这个标签仅可用于电脑应用和浏览器游戏，移动设备无效。

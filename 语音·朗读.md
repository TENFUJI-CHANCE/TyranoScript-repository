### <a id="voconfig"></a> **[voconfig] 播放语音的设定**

该标签用于给角色添加语音，例如配音。该标签将会把语音和角色绑定，从而实现在角色文本显示时自动播放配音的效果。

使用此标签定义语音后，请用[vostart]标签来启用自动播放。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|sebuf|设定用于播放语音的声道|
|name|设定要绑定的角色名称|
|vostorage|设定要使用的音频文件。一点小技巧，通过使用“{number}”来实现批量播放，这意味着你无需为一个人物使用多个[voconfig]标签|
|number|设定vostorage中的{number}部分的初始编号。|
|waittime|设定在自动播放（autoplay）模式下，请设定语音播放结束后等待多少毫秒再转到下一条消息|

代码结构及示例：
```
[voconfig sebuf="1" name="example" vostorage="example_{number}.ogg" number="1" waittime="500"]

#example
play voice example_1.ogg

#example
play voice example_2.ogg
```
  
### <a id="vostart"></a> **[vostart] 自动播放语音**

该标签用于自动播放由[voconfig]设定的语音。当引擎读取到此标签时，不会立刻播放语音，而是在引擎读取到“#”后才会播放。

代码结构及示例：
```
[vostart]
```
  
### <a id="vostop"></a> **[vostop] 停止自动播放语音**

该标签用于停止自动播放设定的语音。当引擎读取到此标签后，后续的语音播放将自动停止。

代码结构及示例：
```
[vostop]
```
  
### <a id="speak_on"></a> **[speak_on] 启用朗读功能（浏览器限定）**

该标签将使用文本转语音功能把故事的情节朗读出来。注意，无法在PC应用上使用。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|volume|设定朗读的音量。数值范围为0到100|100|
|pitch|设定朗读的音调。正常的音调为100，这是音调的基准，高于这个数值音调变高，低于这个数值音调变低|100|
|rate|设定朗读的速度。正常的速度为100，这是速度的基准，高于这个数值朗读变快，低于这个数值朗读变慢|100|
|cancel|当前朗读的文本被新文本打断时，设定是否要停止当前的朗读，朗读新文本。若要在新文本出现后停止上一条文本的朗读，请设定为“true”|false|

代码结构及示例：
```
[speak_on volume="100" pitch="150" rate="50" cancel="false" ]
```
  
### <a id="speak_off"></a> **[speak_off] 关闭朗读功能**

该标签用于停止文本的朗读。

代码结构及示例：
```
[speak_off]
```

<a id="jump"></a>**[jump]剧本跳跃**

设定要跳转的剧本文件，并移动到指定的标记位置。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定要跳转的剧本|
|target|设定要跳转的标记|
代码结构及示例：
```
[jump storage="example.ks" target="*example" ]
```

<a id="link"></a>**[link]超链接(选项分支)**

将文本包含在[link]和[endlink]中，使其成为可点击的链接。这可用于展示选项。通过鼠标或键盘可以选中链接文本。当通过点击或按键输入选中或点击时，它将跳转到特定的场景文件或标签。链接不能跨越多个页面（但可以跨越多行）。在展示选项之后，您必须添加一个“[s]”标签来暂停流程。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定要跳转的剧本|
|target|设定要跳转的标记|
|keyfocus|该选项通过设置数值来自定义手柄或键盘的光标按照何种顺序切换选项的选择，数字越小优先级越高。若设为false，该选项无法通过键盘或手柄进行选择|
代码结构及示例：
```
[link storage="example.ks" target="*example" keyfocus="1" ]
```

<a id="endlink"></a>**[endlink]超链接(选项分支)的结束**

用于结束[link]所标记的文本。请注意，这种类型的按钮仍然会在文本框里，且与普通文本没有特别明显的差别。

<a id="button"></a>**[button]图形按钮的显示**

显示一个图形按钮。[link]是此标签的文本版本。

当显示图形按钮时，情节推进将被强制中断。请务必指定一个跳转目的地。跳转之后，您必须使用 [cm] 标签来移除该按钮。图形按钮的位置遵循之前[locate]标签所设定的位置。然而，如果指定了 x 或 y 属性，它们的优先级将更高。

【对于标准选择按钮（fix=false）】使用此按钮进行跳转时，不会留下调用栈，类似于使用[jump]。这意味着您不能使用[return]标签返回。

【对于固定按钮（fix=true）】与使用[call]类似，它会留下调用栈。这意味着您必须使用[return]标签从调用中返回。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|graphic|设定按钮使用的图像文件，默认在data/image文件路径|
|folder|设定按钮图像所在的文件夹|image|
|storage|设定要跳转的剧本|
|target|设定要跳转的标记|
|name|该变量给[anim]使用，用于让[anim]为其添加动画|
|x|设定按钮的x轴位置|
|y|设定按钮的y轴位置|
|width|设定按钮的宽度|
|height|设定按钮的高度|
|fix|确定该按钮是否应为固定按钮（例如，始终显示的按钮，如保存按钮）|
|role|为按钮分配特定功能，忽略存储和目标设置。同时强制fix=“true”。可用选项：save（打开保存屏幕）；load（打开加载屏幕）；title（返回标题屏幕）；menu（打开菜单）；window（隐藏消息窗口）；skip（启动跳过模式）；backlog（打开过去文本列表）；fullscreen（切换全屏模式）；quicksave（执行快速保存）；quickload（执行快速加载）；auto（启动自动模式）；sleepgame（保存游戏状态并跳转）|
代码结构：
```
[button graphic="example.jpg"	folder="bgimage" storage="example.ks" target="*example" name="example" x="10" y="10" width="10" height="10” fix="true" role="save" ]
```

<a id="glink_config"></a>**[glink_config]图形链接的设定**

该标签版本需求：V515及以上

该标签用于对[glink]自动布局进行设置。

当自动布局功能启用（默认情况下启用）时，任何未指定 x 坐标和 y 坐标的[glink]都将进行自动布局。这些自动布局的[glink]不会立即显示，而是会在到达[s]标签时才显示出来。如果参数未被指定，则其设置将不会发生改变。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|auto_place|[glink]自动布局的启用开关。该属性设为“true”可启用[glink]的自动布局功能（无需指定 x 和 y 值）。将其设为“false”则会关闭该功能|
|auto_place_force|强制已指定 x 和 y 的[glink]自动布局的启用开关。该属性设定为“true”后，即便[glink]设定了 x 和 y 属性，该标签也将进行自动布局|
|margin_x|设定[glink]按钮的水平外边距|
|margin_y|设定[glink]按钮的垂直外边距|
|padding_x|设定[glink]按钮的水平内边距，如果设为“default”，则不会进行任何调整|
|padding_y|设定[glink]按钮的垂直内边距，如果设为“default”，则不会进行任何调整|
|width|设定[glink]按钮的宽度，这个数值是一种通用宽度，意味着如果[gilnk]我没有单独设置width属性的值，那么[glink]将会按照该值设定按钮宽度。如果设置为“max”，按钮宽度将调整为允许的最大宽度。如果设置为“default”，则不会进行任何调整|
|height|设定[glink]按钮的高度，这个数值是一种通用高度，意味着如果[gilnk]我没有单独设置height属性的值，那么[glink]将会按照该值设定按钮高度。如果设置为“max”，按钮宽度将调整为允许的最大高度。如果设置为“default”，则不会进行任何调整|
|vertical|设定按钮文本在垂直方向上的对齐方式。有如下属性：top（顶部）、center（中部）、bottom（底部）|
|horizontal|设定按钮文本在水平方向上的对齐方式。有如下属性：left（左对齐）、center（中部）、right（右对齐）|
|wrap|该属性与place_area属性结合使用，此属性用于判定[glink]按钮无法容纳时是否自动换行。设定值为“wrap”时启用|
|place_area|定义用于对齐按钮的参考区域。如果设置为“auto”，系统会根据程序窗口自动调整。“cover”会将整个屏幕作为参考区域。如果需要特定的值，则应输入四个用逗号分隔的数字，请严格按照“left,top,width,height”的顺序设定此值，不要空着。|auto|
|show_time|显示按钮动画所需的时间。设定为“0”时不启用动画。注意，动画进行时该按钮不可用|
|show_effect|设定显示按钮的动画效果。允许的值有：fadeIn, fadeInDown, fadeInLeft, fadeInRight, fadeInUp, lightSpeedIn, rotateIn, zoomIn, bounceIn, vanishIn, puffIn|
|show_delay|设定每个按钮的出现时间间隔。设定为“0”时，所有按钮将同时出现|
|show_easing|按钮出现动画的缓动函数：ease（起始和结束处平滑过渡）、linear（速度恒定）、ease-in（开始时缓慢）、easeout（结束时缓慢）、ease-in-out（开始和结束时均缓慢）|
代码结构及示例：
```
[glink_config auto_place="true" auto_place_force="true" margin_x="default" margin_y="100" padding_x="100" padding_y="100" width="max" height="100" vertical="top" horizontal="left" wrap="nowrap" place_area="100,100,0,1000" show_time="2000" show_effect="fadeIn" show_delay="200" show_easing="ease" ]
```

<a id="glink"></a>**[glink]图形链接**

您可以显示一个图形链接（文本按钮）。无需提供图片。当显示图形链接时，情节推进将被强制暂停，因此您必须指定一个跳转目的地。图形链接的显示位置将遵循前一个 [locate] 标签所指定的位置。然而，如果指定了 x 和 y 属性，它们将优先使用。

使用此按钮来导航情节时，其功能类似于 [jump] 标签。这意味着它不会留在调用栈中。换句话说，您无法使用 [return] 标签返回。跳过后，[cm] 标签会自动执行，并且按钮会被移除。

在V501c版本更新后，[glink]添加了一些新的元素。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|color|可以按照下列关键词改变按钮颜色：黑色（black）、灰色（gray）、白色（white）、橙色（orange）、红色（red）、蓝色（blue）、绯红（rosy）、绿色（green）、粉色（pink）|black|
|font_color|设定字体颜色，以0xRRGGBB格式|
|storage|设定要跳转的剧本|
|target|设定要跳转的标记|
|name|该变量给[anim]使用，用于让[anim]为其添加动画。注意，该变量可在HTML中当作一个类（class）使用|
|text|设定按钮文本|
|x|设定按钮的x轴位置|auto|
|y|设定按钮的y轴位置|
|width|设定按钮的宽度|
|height|设定按钮的高度|
|size|设定字体大小|30|
|face|设定字体样式。该字体样式在tyrano/css/font.css中定义，这也意味着CSS的字体库也可以用|
|graphic|设定按钮的图片背景。注意，一旦设置了这个属性，color属性将失效|
|enterimg|在设定graphic的情况下使用。该属性设定当光标悬停在按钮上时显示的图像|
|clickse|设定按钮的点击音效。音效文件请放在data/sound里|
|autopos|设定按钮是否自动调整位置|false|

代码结构：
```
color="black" font_color="0xffffff" storage="example.ks" target="*example" name="example" text="example" x="auto" у="0" width="10" height="10" size="30" face="example" graphic="example.jpg" enterimg="example.png" clickse="example.m4a" autopos="false"
```

<a id="clickable"></a>**[clickable]设置可点击区域**

您可以定义一个透明的可点击区域。当显示可点击区域时，情节推进会强制暂停，因此您必须指定一个跳转目的地。

使用此按钮来编排情节时，其功能类似于[jump]标签，这意味着它不会留在调用栈中。换句话说，您无法使用[return]标签返回。

注意：只有在到达[s]标签后，该可点击区域才会激活。请务必使用[s]标签暂停游戏。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|width|设定可点击区域的宽度|0|〇|
|height|设定可点击区域的高度|0|〇|
|x|设定可点击区域的x轴位置|
|y|设定可点击区域的y轴位置|
|borderstyle|设定该区域周围所显示边框的样式。使用格式“ thickness:type:color”。每个项目均应按照 CSS 标记法进行书写。边框类型包括实线（solid）、双线（double）、槽线（groove）、虚线（dashed）、点线（dotted）等|
|color|设定可点击区域的颜色，以0xRRGGBB格式|
|opacity|设定可点击区域的透明度，值的范围为0到255，0为完全透明|140|
|mouseopacity|设定当鼠标位于可点击区域时鼠标的透明度|
|storage|设定要跳转的剧本|
|target|设定要跳转的标记|
代码结构：
```
[clickable width="100" height="100" x="10" y="10" borderstyle="10px:dotted:red" color="0xffffff" opacity="140" mouseopacity="255" storage="example.ks" target="*example"]
```

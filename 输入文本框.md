### <a id="edit"></a> **[edit] 添加可供玩家输入的文本框**

该标签会在游戏里显示一个文本框，玩家可以在此输入文本。当该文本框出现时，游戏情节的进度将会暂停。请务必放置一个图形按钮或类似的元素，以便玩家能够继续操作。

输入的文本将会存储在指定的变量中，当执行[commit]标签时会进行存储。[commit]标签必须在文本框仍可见的情况下执行。

当与[glink]标签结合使用时，请在[glink]标签中指定cm=false。

高级用法：<br />文本框的设计可以通过编辑“tyrano/tyrano.css”文件来进行调整。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定用于保存玩家输入文本的变量名称||〇|
|initial|设定文本框中初始显示的文本内容|
|color|设定文本框内的文本颜色|black|
|left|设定文本框的x轴位置|0|
|top|设定文本框的y轴位置|0|
|autocomplete|设定是否显示输入历史记录|false|
|size|设定文本的字体大小|20|
|width|设定文本框的宽度|200|
|height|设定文本框的高度|40|
|maxchars|设定文本框中允许输入的最大字符数|1000|

代码结构及示例：
```
[edit name="player_input" initial="请输入文本" color="black" left="0" top="0" autocomplete="false" size="20" width="200" height="40" maxchars="1000"]
```
  
### <a id="commit"></a> **[commit] 将玩家输入的内容转化为程序用变量**

该标签将根据[edit]中显示的文本框输入内容，将输入内容变为由name属性指定的变量的值。

这就意味着，当玩家在文本框中输入内容并执行[commit]标签后，输入的内容被保存到指定的变量中才可使用。

代码结构及示例：
```
[commit name="player_input"]
```

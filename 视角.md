### <a id="camera"></a> **[camera] 移动相机**

您可以利用相机功能实现诸如缩放和平移等效果。通过使用相机功能，您可以聚焦于角色的表情，或者将镜头拉近到插图的不同部分以产生各种效果。

要使用相机功能，您必须在Config.tjs文件中将useCamera属性设置为true。此外，当启用相机功能时，屏幕居中功能ScreenCentering将被禁用。

相机坐标是以屏幕中心为基准进行设定的，即游戏屏幕正中央为(0，0)。例如，屏幕的右上角为（200，200），而左下角为（-200，-200）。

如果没有指定from_x、from_y、from_zoom或from_rotate属性，则相机将从当前的位置开始移动。

要将相机重置到初始位置，请使用 [reset_camera] 标签。要等待相机效果完成，请使用 [wait_camera] 标签。

**注意：在相机效果完成后，请务必使用 [reset_camera] 标签将相机位置重置为初始位置。如果不进行重置，您将无法使用可以进行背景变化的[bg]标签等。**

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定相机移动的时间|1000|
|x|设定相机移动的目标x坐标。设置该属性后，相机将移动到指定的坐标|1000|
|y|设定相机移动的目标y坐标。设置该属性后，相机将移动到指定的坐标|
|zoom|设定相机移动的目标缩放倍数。设置该属性后，相机将缩放指定倍数|
|rotate|设定相机移动的目标旋转角度。设置该属性后，相机将旋转指定角度|
|from_x|设定相机移动的起始x坐标。设置该属性后，相机将从指定的坐标开始移动|
|from_y|设定相机移动的起始y坐标。设置该属性后，相机将从指定的坐标开始移动|
|from_zoom|设定相机移动的起始缩放倍数。设置该属性后，相机将从指定的缩放倍数开始缩放|
|from_rotate|设定相机移动的起始旋转角度。设置该属性后，相机将从指定的旋转角度开始旋转|
|wait|设定是否等待相机移动完成|true|
|layer|指定应用相机效果的图层。通过指定此选项，可以将相机效果限制在特定的图层上，例如背景层（base）。通常情况下，该效果会作用于所有图层|layer_camera|
|ease_type|设定相机的移动效果。 ease（从开始到结束的平滑过渡） linear（匀速过渡） ease-in（开始时缓慢） ease-out（结束时缓慢） ease-in-out（开始和结束时都缓慢）|

代码结构及示例：
```
[camera time="1000" x="100" y="100" zoom="2" rotate="45" wait="true" layer="base" ease_type="ease-in-out"]
```
  
### <a id="reset_camera"></a> **[reset_camera] 重置相机**

该标签将重置相机位置到初始状态。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定相机移动的时间|1000|
|wait|设定是否等待相机移动完成|true|
|layer|指定应用相机效果的图层。通过指定此选项，可以将相机效果限制在特定的图层上，例如背景层（base）。通常情况下，该效果会作用于所有图层|layer_camera|
|ease_type|设定相机的移动效果。 ease（从开始到结束的平滑过渡） linear（匀速过渡） ease-in（开始时缓慢） ease-out（结束时缓慢） ease-in-out（开始和结束时都缓慢）|

代码结构及示例：
```
[reset_camera time="1000" wait="true" layer="base" ease_type="ease-in-out"]
```
  
### <a id="wait_camera"></a> **[wait_camera] 等待相机演出**

如果摄像机正在执行某种效果，该标签将等待其完成。

可以将“wait=false”与[camera]标签结合使用。例如，您可以将其用于这样一个场景：“在显示文字的同时，摄像机在背景中移动。但在转到下一条消息时，要等待摄像机完成移动。”

代码结构及示例：
```
[wait_camera]
```

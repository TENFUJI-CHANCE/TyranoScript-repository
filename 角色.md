### <a id="chara_ptext"></a> **[chara_ptext] 角色名字表示与表情变更**

该标签用于设定显示角色名字和角色表情。在使用这个标签之前，需要先用[ptext]设置显示角色的文本，并用[chara_config]绑定。

该标签可以简写。用“#”表示，后面跟角色名字；在名字之后加上“:”和表情名字，便可修改角色表情。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要显示的角色名字。显示的名字是[chara_new]定义的角色里的jname属性，如果没有jname，就显示赋值给name的文本本身|
|face|设定要显示的角色表情|

代码结构及示例：
```
[chara_ptext name="akane" face="default" ]

#akane:default
```

### <a id="chara_config"></a> **[chara_config] 角色操作标签的基本设置**



属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|pos_mode|此参数用于指示[chara_show]标签是否自动计算角色的所在位置|
|ptext|该属性参数为显示角色名称所用的[ptext]标签创建的文本区域的名称，该参数与[chara_ptext]标签密切相关|
|time|当角色更改面部表情时，其渐隐动画的时间，其与[chara_mod]标签相关|
|memory|此参数用于是否记录游戏关闭时的表情|
|anim|当pos_mode参数的值为true时，是否对角色的位置移动动画化|
|pos_change_time|自动调整角色位置时的动画的时间|
|talk_focus|该参数用于使当前正在说话的角色的立绘图突出显示，可用值为brightness（高亮）、blur（模糊）、none（无）|
|brightness_value|当talk_focus参数为brightness时，此参数用于确定高亮的亮度，这将使未说话的角色的亮度有所降低|
|blur_value|当talk_focus参数为blur时，此参数用于确定模糊度（默认为2），这将使未说话的角色立绘图变得模糊|
|talk_anim|角色开始说话的时候，该参数用于添加角色的立绘自动跳跃般的动画演出的设定。可用值为up（向上）、down（向下）、zoom（放大）、none（无影响）|
|talk_anim_time|此参数用于设定talk_anim设定的动画的经过时间（单位为毫秒）|
|talk_anim_value|当talk_anim的值非none时，角色的移动距离|
|talk_anim_zoom_rate|当talk_anim的值为zoom时，此参数设定放大倍数，默认为1.2|
|effect|设定角色调换位置时的效果(移动方式)，移动方式由以下关键词设置：<br />`jswing`、`def`、`easeInQuad`、`easeOutQuad`、`easeInOutQuad`、`easeInCubic`、`easeOutCubic`、`easeInOutCubic`、`easeInQuart`、`easeOutQuart`、`easeInOutQuart`、`easeInQuint`、`easeOutQuint`、`easeInOutQuint`、`easeInSine`、`easeOutSine`、`easeInOutSine`、`easeInExpo`、`easeOutExpo`、`easeInOutExpo`、`easeInCirc`、`easeOutCirc`、`easeInOutCirc`、`easeInElastic`、`easeOutElastic`、`easeInOutElastic`、`easeInBack`、`easeOutBack`、`easeInOutBack`、`easeInBounce`、`easeOutBounce`、`easeInOutBounce`|

代码结构及示例：
```
[chara_config pos_mode="true" ptext="chara_name" time="1000" memory="true" anim="false" pos_change_time="1000" talk_focus="brightness" brightness_value="0.5" blur_value="2" talk_anim="up" talk_anim_time="500" talk_anim_value="10" talk_anim_zoom_rate="1.2" effect="easeInOutQuad" ]
```

### <a id="chara_new"></a> **[chara_new] 定义角色**

该标签用于注册（定义）角色。

通过使用此标签定义一个角色，您可以使用[chara_show]标签来显示该角色，或者使用[chara_mod]标签来更改该角色的表情。

在此处指定的name参数将在诸如[chara_show]这样的角色操作标签中使用。还可以在诸如 [anim] 或 [kanim] 等标签中使用，从而能够自由地为角色设定动画。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定独一无二的角色ID。该名称必须是唯一的（即不能与任何其他字符的名称相同）。此外，它也不能与[ptext][image]标签的名称重叠||〇|
|storage|设定角色图像。该图像将会在表情中的名称定义为default||〇|
|width|设定角色图像的宽度|
|height|设定角色图像的高度| 
|reflect|设定是否要水平翻转图像。如果设定为true，则角色图像水平翻转|false|
|color|设定用于显示的角色名字颜色，以0xRRGGBB格式|
|jname|设定用于显示的角色名|

代码结构及示例：
```
[chara_new name="yuko" storage="yuko.png" width="10" height="10" reflect="false" color="0xffffff" jname="ゆうこ"]
```

### <a id="chara_show"></a> **[chara_show] 角色上场**

该标签用于让已经定义的角色显示（登场）。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要显示的角色||〇|
|time|设定角色上场的所需时间|1000|
|layer|设定角色所在的图层|0|
|zindex|设定角色的堆叠顺序。数值越大，该角色在屏幕上的显示位置就越靠前。若未指定，则后续出现的角色将显示在前面。|1|
|depth|如果zindex值相同，则需指定堆叠顺序为“front”（最前面）或“back”（最后面）。|front|
|page|设定角色所在是在前景还是背景|fore|
|wait|设定是否等待角色上场完毕|true|
|face|设定要显示的角色表情|
|storage|设定要显示的角色图像|
|reflect|设定是否要水平翻转图像。如果设定为true，则角色图像水平翻转|
|width|设定角色图像的宽度|
|height|设定角色图像的高度|
|left|设定角色的x轴位置|
|top|设定角色的y轴位置|

代码结构及示例：
```
[chara_show name="yuko" time="1000" layer="0" zindex="1" depth="front" page="fore" wait="true" face="default" storage="yuko.png" reflect="false" width="10" height="10" left="0" top="0" ]
```

### <a id="chara_hide"></a> **[chara_hide] 角色退场**

该标签用于将使用[chara_show]显示的角色隐藏（退场）。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要退场的角色||〇|
|time|设定角色退场的所需时间|1000|
|wait|设定是否等待角色退场完毕|true|
|layer|设定要删除角色的图层。如果[chara_show]设定了图层，那么这里也要设定|0|
|pos_mode|当角色的调整位置设定为自动时，如果该参数为false，则其他已应用自动调整位置设定的角色不会自动调整|true|

代码结构及示例：
```
[chara_hide name="yuko" time="1000" wait="true" layer="0" pos_mode="true" ]
```

### <a id="chara_mod"></a> **[chara_mod] 角色图像变更**

该标签用于改变角色图像，经常用于改变角色表情。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要改变表情的角色||〇|
|face|设定要改变的表情|
|time|设定渐隐动画所需的时间|
|storage|设定所要切换的角色图像|
|reflect|设定是否要镜像翻转图像|
|wait|设定是否等待切换完成|true|
|cross|设定是否采用“渐隐”动画的方法切换表情。若选择“true”，则会先将旧图像渐隐，然后逐渐叠加新图像；若选择“false”，则会将新图像叠加在旧图像之上，而不会让旧图像渐隐。若将该选项设为“true”，可能会出现角色部分透明的瞬间，从而使得背景得以显现。在这种情况下，将其设为“false”可避免在表情变化时出现透明度现象。然而，如果表情变化改变了轮廓，可能会显得不自然。|true|

代码结构及示例：
```
[chara_mod name="yuko" face="angry" time="1000" storage="yuko_angry.png" reflect="false" wait="true" cross="true" ]
```

### <a id="chara_move"></a> **[chara_move] 角色位置移动的变更**

该标签用于改变角色的位置。与[chara_show]标签不同的是，使用[chara_move]标签时，角色必须已经显示（登场）。如果要改变角色的位置，请使用此标签。

靠着这个标签你可以自定义一个多样化的登场动画。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要移动的角色||〇|
|left|指定移动后的横向位置，如果值的设定采用“+=（或-=）(数值)”的形式，则该设定意义为相对移动量（增加量或减少量）|
|top|指定移动后的纵向位置，此参数也可使用“+=”和“-=”|
|width|设定移动后角色的宽度|
|height|设定移动后角色的高度|
|anim|设定移动是否用动画形式，动画效果参照[chara_config]的effect参数|false|
|time|设定移动动画时间|600|
|wait|设定是否等待动画播放完毕|true|
|effect|设定移动动画效果。有如下可选动画：<br />`jswing`、`def`、`easeInQuad`、`easeOutQuad`、`easeInOutQuad`、`easeInCubic`、`easeOutCubic`、`easeInOutCubic`、`easeInQuart`、`easeOutQuart`、`easeInOutQuart`、`easeInQuint`、`easeOutQuint`、`easeInOutQuint`、`easeInSine`、`easeOutSine`、`easeInOutSine`、`easeInExpo`、`easeOutExpo`、`easeInOutExpo`、`easeInCirc`、`easeOutCirc`、`easeInOutCirc`、`easeInElastic`、`easeOutElastic`、`easeInOutElastic`、`easeInBack`、`easeOutBack`、`easeInOutBack`、`easeInBounce`、`easeOutBounce`、`easeInOutBounce`|

代码结构及示例：
```
[chara_move name="yuko" left="100" top="100" width="200" height="200" anim="true" time="1000" wait="true" effect="easeInOutQuad" ]
```

### <a id="chara_layer"></a> **[chara_layer] 定义角色的差分部分**

该标签用于定义角色的差分部分，请注意，素材必须在图片大小上一致。使用该标签时不用担心定位问题，因为定位是由[chara_show]实现的。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要应用差分的角色||〇|
|part|请指定要注册的部件名称。可以注册多个差异项，也就是storage可以不同。||〇|
|id|设定差分的ID。该ID将在[chara_part]标签中使用，以指定要更改的差分项。ID不一定是数字，也可以是字符||〇|
|storage|设定差分图像，图像文件应位于data/fgimage中|
|zindex|设定差分的层级，数值越大层级越高，也就是越容易靠前显示|

代码结构及示例：
```
[chara_layer name="yuko" part="eye" id="normal" storage="example.png" zindex="3"]
```

### <a id="chara_part"></a> **[chara_part] 变更角色差分的部分**

[chara_layer]标签会改变所定义部分差异的实际显示效果。此标签的参数设置方式较为特殊。也就是将part的值作为变量，id为该变量的可选值。形成“[part]=[id]”这样的格式。比如，如果part的值是“eye”，id的值是“happy”，则可以通过设置“eye=happy”来改变角色的眼睛部分为“happy”表情。

可以同时更改多个部分。无需指定ID，可以直接指定差异图像文件。在这种情况下，请指定allow_storage=true。

还可以通过为名为“part_name_zindex”的参数指定一个数值来更改特定部分的 z 值。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要改变部件的角色||〇|
|time|设定改变部件所需的时间|
|wait|设定是否等待部件改变完成|true|
|allow_storage|是否可直接指定图像文件，而无需使用部件的标识符|false|

代码结构及示例：
```
[chara_part name=yuko mouse=happy eye=happy]
```

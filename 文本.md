###<a id="l"></a>**[l] 等待点击**

当引擎读取到此标签时，游戏将会等待玩家点击，玩家点击后下一句将会出现。

###<a id="p"></a>**[p] 等待点击+改页**

当引擎读取到此标签时，游戏将会等待玩家点击，点击后消息文本会清空后显示下一句。

###<a id="r"></a>**[r] 换行**

当引擎读取到此标签时，文本会自动换行。

###<a id="er"></a>**[er] 删除消息层的文字**

当引擎读取到此标签时，引擎会清空当前消息层里的文本并显示下一句。

###<a id="cm"></a>**[cm] 重置消息层**

清除消息层里所有含有文本的元素，字体也将回到初始状态。

###<a id="ct"></a>**[ct] 重置消息层状态**

该标签在[cm]的基础上将切换当前消息层到初始消息层。

设置该标签是因为引擎本身的消息层是这样工作的：

当设置两个及以上的消息层时，引擎只确保一个消息层工作（也就是只保证一个消息层可以点击操作，使用[current]切换消息层），[ct]的目的在于，防止作者在使用[cm]后忘记了当前引擎所操作的消息层。

###<a id="current"></a>**[current] 指定操作对象的消息层**

该标签可以改变引擎所要操作的消息层。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|layer|指定要操作的消息层|
|page|指定消息层要操作的是前景还是背景|fore|

代码结构及示例：
```
[current layer="message0" page="fore" ]
```

###<a id="graph"></a>**[graph] 使图像在文本中显示**

该标签用于将图像插入到消息层中。请将图片放进文件路径为data/image的文件夹里。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|指定要使用的图片|

代码结构及示例：
```
[graph storage="example.jpg" ]
```

###<a id="fuki_start"></a>**[fuki_start] 将文本转化成气泡对话框**

这个标签的作用是把消息层转变成对话气泡的形式，这样增加了视觉上的动态变化。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|layer|指定要转换成气泡对话框的消息层|message0|

代码结构及示例：
```
[fuki_start layer="message0" ]
```

###<a id="fuki_stop"></a>**[fuki_stop] 取消气泡对话框的渲染**

这个标签将把对话气泡改回应用对话气泡之前的状态。

###<a id="fuki_chara"></a>**[fuki_chara] 注册使用对话气泡的角色**

该标签用于将对话气泡与角色绑定，这是动态的，使用该标签后，无论对话气泡在什么位置，都会有一个小三角指向角色。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|设定要绑定的角色，此变量值为由[chara_new]定义的name属性值，当没有指定角色时，请使用other代替||〇|
|left|设定对话框的x轴位置，该位置是相对于角色位置来说的，也就是说，是以角色定位点为原点|
|top|设定对话框的y轴位置，该位置是相对于角色位置来说的，也就是说，是以角色定位点为原点|
|sippo|设定气泡指向说话角色的方向（一般称之为尾巴方向），可选的值为top（上）, bottom（下）, left（左）, right（右）|top|
|sippo_left|设定气泡尾巴的横向位置，该属性是用于sippo值为top或bottom时|40|
|sippo_top|设定气泡尾巴的纵向位置，该属性是用于sippo值为left或right时|40|
|sippo_width|设定对话气泡的宽度|12
|sippo_height|设定对话气泡的高度|20|
|max_width|当对话气泡的宽度自动调整时，最长的宽度|300|
|fix_width|停止自动调整对话气泡的宽度，并将对话框宽度固定为指定值。|
|color|设定对话气泡的背景颜色，以0xRRGGBB格式|
|border_color|设定对话气泡的边框颜色|
|border_size|设定对话气泡的边框粗细，值为0时不显示边框|0|
|opacity|设定对话气泡的透明度，值的范围为0到255，0为完全透明。注意这个不影响文本和图层透明度|
|radius|设定对话气泡的圆角。有如下参考值：10（略微圆润型）；30（中等圆润型）；100（高度圆润型）|
|font_color|设定字体颜色，以0xRRGGBB格式|
|font_size|设定字体大小|

代码结构及示例：
```
[fuki_chara name="example" left="10" top="10" sippo="top" sippo_left="40" sippo_top="40" sippo_width="12" sippo_height="20" max_width="300" fix_width="300" color="0xffffff" border_color="0xffffff" border_size="0" opacity="255" radius="30" font_color="0xffffff" font_size="10" ]
```

###<a id="ptext"></a>**[ptext] 在图层中显示文本**

该标签用于让文本在前景层中显示。要想移除该标签设置的文本，请使用[freeimage]标签。默认状态下该图层不显示，需要使用[layopt]让该图层显示。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|layer|设定要让文本显示的图层|0|〇|
|page|设定要显示文本的是在图层前景还是背景|fore|
|text|设定要显示的文本内容|
|x|设定文本的x轴位置||〇|
|y|设定文本的y轴位置||〇|
|vertical|设定文本是否纵向排列|false|
|size|设定字体大小|
|face|设定字体样式|
|color|设定字体颜色，以0xRRGGBB格式|
|bold|设定字体是否加粗，在V515版本之后使用true也可加粗|
|edge|设定文本边框，使用0xRRGGBB格式设定颜色，空值表示不启用，在V515版本之后，按照2px 0xffffff的格式可以设定边框粗细和颜色，粗细由像素px设定，颜色以0xRRGGBB格式设定|
|shadow|设定是否使用阴影，使用0xRRGGBB格式设定颜色，空值表示不启用|
|name|该变量给[anim]使用，用于让[anim]为其添加动画|
|width|设定文本框的宽度|
|align|设定文本对齐方式，值为left（左对齐）, center（居中）, right（右对齐）|left|
|time|设定文本渐变淡入时间|
|overwrite|当遇到相同的name属性的文本时，设定是否启用覆盖|false|
|gradient|V515版本之后，设定文本使用的渐变效果，空值表示不启用，该值使用CSS的linear-gradient()函数|

代码结构及示例：
```
[ptext layer="0" page="fore" text="example" x="10" y="10" vertical="false" size="10" face="example" color="0xffffff" bold="true" edge="2px 0xffffff" shadow="0xffff00" name="example" width="10" align="left" time="1000" overwrite="false" gradient="linear-gradient(45deg, red 0 50%, blue 50% 100%)" ]
```

###<a id="mtext"></a>**[mtext] 有特殊动画效果的文本**

在屏幕上以各种动画效果显示文本。默认状态下该图层不显示，需要使用[layopt]让该图层显示。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|layer|设定要让文本显示的图层|0|
|page|设定要显示文本的是在图层前景还是背景|fore|
|text|设定要显示的文本内容|
|x|设定文本的x轴位置||〇|
|y|设定文本的y轴位置||〇|
|vertical|设定文本是否纵向排列|false|
|size|设定字体大小|
|face|设定字体样式|
|color|设定字体颜色，以0xRRGGBB格式|
|width|设定文本框的宽度|
|align|设定文本对齐方式，值为left（左对齐）, center（居中）, right（右对齐）|left|
|name|该变量给[anim]使用，用于让[anim]为其添加动画|
|bold|设定字体是否加粗|
|edge|设定文本边框，使用0xRRGGBB格式设定颜色，空值表示不启用|
|shadow|设定是否使用阴影，使用0xRRGGBB格式设定颜色，空值表示不启用|
|fadeout|是否在显示文本后执行淡出效果使其消失|true|
|time|设定文本动画的持续时间（以毫秒为单位）|2000|
|wait|是否暂停游戏等待动画完成|true|
|in_effect|设定文本出现时的动画效果|fadeIn|
|out_effect|设定文本消失时的动画效果|fadeOut|

代码结构及示例：
```
[mtext layer="0" page="fore" text="example" x="10" y="10" vertical="false" size="10" face="example" color="0xffffff" width="10" align="left" name="example" bold="bold" edge="0xffffff" shadow="0xffff00" fadeout="true" time="2000" wait="true" in_effect="fadeIn" out_effect="fadeOut"]
```

###<a id="ruby"></a>**[ruby] 文字注音**

该标签用于给文本添加注音。注意，这个标签是给标签后面的字注音，且只有一个字。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|text|设定注音要用的文本||〇|

代码结构及示例：
```
[ruby text="example" ]例子
```

###<a id="mark"></a>**[mark] 文本标记**

该标签用于给文本做底纹标记。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|color|设定底纹颜色，以0xRRGGBB格式|0xFFFF00|
|font_color|设定字体颜色，以0xRRGGBB格式|
|size|设定标记的大小，范围为 1 到 100。当该变量值为 50 会使文本的下半部分突出显示，而 10 则会使其效果类似于下划线|

代码结构及示例：
```
[mark color="0xffff00" font_color="0xffffff" size="10" ]
```

###<a id="endmark"></a>**[endmark] 结束文本标记**

该标签用于结束[mark]开始的文本底纹标记，可以说，这是一个用于划定文本范围的“句号”。

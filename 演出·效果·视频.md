### <a id="quake"></a> **[quake] 晃动画面效果**

该标签用于让游戏画面抖动。该标签可以实现垂直抖动和水平抖动，当hmax为0，vmax不为0时为垂直抖动，反之则是水平抖动。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|count|设定抖动次数|5|
|wait|设定是否等待抖动结束|true|
|time|设定一次抖动所需的时间|300|〇|
|hmax|设定抖动时x轴方向的最大值|0|
|vmax|设定抖动时y轴方向的最大值|10|

代码结构及示例：
```
[quake count="5" wait="true"]
```
  
### <a id="quake2"></a> **[quake2] 晃动画面效果**

该标签用于让游戏画面抖动。与[quake]标签不同的是，该标签不必强制设定单次抖动时间。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定游戏画面抖动的全部时间|1000|
|hmax|设定抖动时x轴方向的最大值|0|
|vmax|设定抖动时y轴方向的最大值|200|
|wait|设定是否等待抖动结束|true|
|copybase|设定此属性为“true”，在屏幕抖动时，底层背景的会复制并固定在最底层。这能防止屏幕向上抖动时底部出现黑色空隙|true|

代码结构及示例：
```
[quake2 time="1000" hmax="0" vmax="200" wait="true" copybase="true"]
```
  
### <a id="vibrate"></a> **[vibrate] 手机（游戏手柄）振动**

该标签用于使手机和游戏手柄振动。

如果指定的振动时间过长，可能会停止振动，所以要谨慎。根据环境的不同，通常建议将振动时间控制在5000毫秒以内。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定振动的持续时间。还有一种特别的振动模式：`600,200,1000,200,600`这样的数值指定了一个模式，即振动 600 毫秒→静止 200 毫秒→静止 1000 毫秒→……|500|
|power|设定振动强度（0 到 100）。此参数仅在对游戏手柄进行振动操作时有效|100|
|count|设定重复振动的次数|

代码结构及示例：
```
[vibrate time="500" power="100" count="1"]
```
  
### <a id="vibrate_stop"></a> **[vibrate_stop] 停止手机（游戏手柄）振动**

该标签用于使手机和游戏手柄停止振动。此标签的目的是在振动时间结束前强行停止。

代码结构及示例：
```
[vibrate_stop]
```
  
### <a id="layermode"></a> **[layermode] 混合图层模式**

您可以让一幅图像变成一种混合模式滤镜，从而覆盖在游戏画面上。

混合方式的种类多样，如正片叠底、屏幕叠加、重叠等。但某些较旧的浏览器（如IE）可能无法支持此功能，因此如果将其作为浏览器游戏发布，需谨慎操作。建议将其用于作为PC应用程序输出的游戏中。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|为混合的图像命名一个名称。此名称可在 [free_layermode] 标签中使用，以便在需要移除特定合成图像时进行引用|
|graphic|设定要用于混合的图像。图像文件应放在image文件夹里|
|color|可以通过设定颜色的方式来替代图像。颜色格式为0xRRGGBB十六进制|
|mode|设定混合模式。<br />以下是可用的模式选项：`multiply`（正片叠底）、`screen`（滤色）、`overlay`（叠加）、`darken`（变暗）、`lighten`（变亮）、`color-dodge`（颜色减淡）、`color-burn`（颜色加深）、`hard-light`（强光）、`soft-light`（柔光）、`difference`（差值）、`exclusion`（排除）、`hue`（色相）、`saturation`（饱和度）、`color`（颜色）、`luminosity`（明度）|multiply|
|folder|设定图像所在的文件夹。默认为"image"文件夹|
|opacity|设定混合图像的透明度。数值范围为0到255，0表示完全透明|
|time|设定淡入时间（以毫秒为单位）。若设定了该时间，图像将以淡入动画的形式逐渐显现出来。若未指定，则会立即出现|500|
|wait|设定是否等待淡入动画完成后再继续|true|

代码结构及示例：
```
[layermode name="overlay1" graphic="overlay.png" mode="overlay" opacity="128" time="500" wait="true"]
```
  
### <a id="layermode_movie"></a> **[layermode_movie] 混合图层模式（视频）**

与[layermode]标签类似，但使用视频作为混合图像。请注意，使用视频可能会对性能产生较大影响，因此建议谨慎使用。

推荐使用 mp4 格式。ogv和webm格式也予以支持。

在将游戏导出为浏览器游戏时，请注意某些视频格式可能不被某些浏览器所支持。具体而言，webm格式在Safari浏览器中无法使用。此外，mp4格式在Firefox和Opera浏览器中也无法使用。在这种情况下，如果在同一位置存在同名的webm文件h和mp4文件，系统将自动选择webm文件。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|为混合的图像命名一个名称。此名称可在 [free_layermode] 标签中使用，以便在需要移除特定合成图像时进行引用|
|video|设定要用于混合的视频。视频文件应放在video文件夹里|
|volume|设定视频的音量。数值范围为0到100|
|mute|设定是否要静音视频。在浏览器中，播放视频之前需要用户进行操作（如点击），但选择“true”则可绕过此限制|false|
|loop|设定是否要循环播放视频。如果设置为循环播放，视频将一直有效，直到调用[free_layermode]标签为止|true|
|speed|设定视频的播放速度。2表示以两倍速播放，0.5表示以半速播放|
|mode|设定混合模式。<br />以下是可用的模式选项：`multiply`（正片叠底）、`screen`（滤色）、`overlay`（叠加）、`darken`（变暗）、`lighten`（变亮）、`color-dodge`（颜色减淡）、`color-burn`（颜色加深）、`hard-light`（强光）、`soft-light`（柔光）、`difference`（差值）、`exclusion`（排除）、`hue`（色相）、`saturation`（饱和度）、`color`（颜色）、`luminosity`（明度）|multiply|
|folder|设定图像所在的文件夹。默认为"image"文件夹|
|opacity|设定混合图像的透明度。数值范围为0到255，0表示完全透明|
|time|设定淡入时间（以毫秒为单位）。若设定了该时间，图像将以淡入动画的形式逐渐显现出来。若未指定，则会立即出现|500|
|left|设定混合图像的x轴位置|
|top|设定混合图像的y轴位置|
|width|设定混合图像的宽度|
|height|设定混合图像的高度|
|fit|设定是否拉伸以适应游戏屏幕|true|
|wait|设定是否等待淡入动画完成后再继续|false|

代码结构及示例：
```
[layermode_movie name="overlay_video" video="overlay.mp4" mode="overlay" opacity="128" time="500" wait="false"]
```

### <a id="free_layermode"></a> **[free_layermode] 清除混合图层模式**

移除之前通过[layermode]或[layermode_movie]标签创建的混合图层模式。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|name|指定要移除的混合图层模式的名称。如果未指定，则会移除所有混合图层模式|
|time|设定淡出时间（以毫秒为单位）。如果设定了该时间，混合图层模式将以淡出动画的形式逐渐消失。若未指定，则会立即消失|500|
|wait|设定是否等待淡出动画完成后再继续|true|

代码结构及示例：
```
[free_layermode name="overlay1" time="500" wait="true"]
```

### <a id="movie"></a> **[movie] 播放视频**

该标签用于在游戏全屏幕播放视频。

推荐使用 mp4 格式。ogv和webm格式也予以支持。

在将游戏导出为浏览器游戏时，请注意某些视频格式可能不被某些浏览器所支持。具体而言，webm格式在Safari浏览器中无法使用。此外，mp4格式在Firefox和Opera浏览器中也无法使用。在这种情况下，如果在同一位置存在同名的webm文件h和mp4文件，系统将自动选择webm文件。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定视频文件的存储位置。默认为"video"文件夹||〇|
|skip|设定是否允许玩家在视频播放过程中跳过视频。如果设定为“true”，则允许玩家通过点击屏幕跳过视频|true|
|volume|设定视频的音量。数值范围为0到100|
|mute|设定是否要静音视频。在浏览器中，播放视频之前需要用户进行操作（如点击），但选择“true”则可绕过此限制|false|

代码结构及示例：
```
[movie storage="video.mp4" skip="true" volume="100" mute="false"]
```
  
### <a id="bgmovie"></a> **[bgmovie] 播放背景视频**

该标签用于在游戏的背景层播放视频。

[stop_bgmovie]会在视频播放完毕后自动停止播放。<br />[bgmovie]可以通过在循环播放过程中使用另一个[bgmovie]来进行分层处理，这样当前视频播放完毕后，下一个视频就会随之开始播放。

请注意这个标签仅限于PC。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|storage|设定视频文件的存储位置。默认为"video"文件夹||〇|
|time|设定淡入时间（以毫秒为单位）。若设定了该时间，图像将以淡入动画的形式逐渐显现出来。|300|
|volume|设定视频的音量。数值范围为0到100|
|mute|设定是否要静音视频。在浏览器中，播放视频之前需要用户进行操作（如点击），但选择“true”则可绕过此限制|false|
|loop|设定是否要在背景层循环播放视频。如果设定为“false”，那么视频将会在最后一帧停止播放|true|

代码结构及示例：
```
[bgmovie storage="background.mp4" time="300" volume="100" mute="false" loop="true"]
```

### <a id="filter"></a> **[filter] 滤镜效果**



属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|layer|设定要应用该滤镜的层。若未指定层或指定了“all”则表示效果将应用于整个游戏屏幕|all|
|name|设定要应用特定滤镜元素的名称|
|grayscale|为图层添加灰度滤镜。设定0到100的数值改变滤镜程度|0|
|sepia|为图层添加棕褐色滤镜。设定0到100的数值改变滤镜程度|0|
|saturate|为图层调整饱和度（鲜艳度）。设定0到100的数值改变滤镜程度|100|
|hue|改变图层的色调。设定0到360的数值改变滤镜程度|0|
|invert|反转图层的色调。设定0到100的数值改变滤镜程度|0|
|opacity|改变图层的透明度。设定0到100的数值改变滤镜程度|100|
|brightness|改变图层的明亮度。当属性值等于0时图像变黑，而大于100的数值则会使图像变亮|100|
|contrast|改变图层的对比度。设定0到100的数值改变滤镜程度|100|
|blur|为图层添加模糊滤镜。数值在0及以上|0|

代码结构及示例：
```
[filter layer="all" grayscale="50" sepia="20" saturate="150" hue="30" invert="10" opacity="80" brightness="120" contrast="110" blur="5"]
```
  
### <a id="free_filter"></a> **[free_filter] 清除滤镜效果**

设定要去除由[filter]标签创建的滤镜效果的图层或对象。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|

|name|设定要去除的特定元素的滤镜，该属性的值为[filter]标签中指定的name值|

代码结构及示例：

  
### <a id="mask"></a> **[mask] 遮罩效果**

该标签使用多种效果为游戏屏幕添加遮罩。在添加遮罩后，可以重新构建游戏屏幕并使用“[mask_off]”标签来恢复游戏。

一般来说，最常用的场景是使游戏屏幕变暗。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定出现遮罩的动画的时间，单位为毫秒|1000|
|effect|设定遮罩出现的动画效果。<br />可用动画效果如下：`fadeIn`、`fadeInDownBig`、`fadeInLeftBig`、`fadeInRightBig`、`fadeInUpBig`、`flipInX`、`flipInY`、`lightSpeedIn`、`rotateIn`、`rotateInDownLeft`、`rotateInDownRight`、`rotateInUpLeft`、`rotateInUpRight`、`zoomIn`、`zoomInDown`、`zoomInLeft`、`zoomInRight`、`zoomInUp`、`slideInDown`、`slideInLeft`、`slideInRight`、`slideInUp`、`bounceIn`、`bounceInDown`、`bounceInLeft`、`bounceInRight`、`bounceInUp`、`rollIn`|fadeIn|
|color|设定遮罩的颜色。颜色格式为0xRRGGBB十六进制|0x000000|
|graphic|该属性可以把一张图像转化成遮罩。图像文件应放在image文件夹里|
|folder|设定图像所在的文件夹。默认为"image"文件夹|

代码结构及示例：
```
[mask time="1000" effect="fadeIn" color="0x000000"]
```
  
### <a id="mask_off"></a> **[mask_off] 清除遮罩效果**

该标签用于清除由[mask]标签创建的遮罩效果。可以设定动画时间和动画效果来淡出遮罩。

属性：
|变量名|解释|默认值|必需|
|-|-|-|-|
|time|设定遮罩消失的动画的时间，单位为毫秒|1000|
|effect|设定遮罩消失的动画效果。<br />可用动画效果如下：`fadeOut`、`fadeOutDownBig`、`fadeOutLeftBig`、`fadeOutRightBig`、`fadeOutUpBig`、`flipOutX`、`flipOutY`、`lightSpeedOut`、`rotateOut`、`rotateOutDownLeft`、`rotateOutDownRight`、`rotateOutUpLeft`、`rotateOutUpRight`、`zoomOut`、`zoomOutDown`、`zoomOutLeft`、`zoomOutRight`、`zoomOutUp`、`slideOutDown`、`slideOutLeft`、`slideOutRight`、`slideOutUp`、`bounceOut`、`bounceOutDown`、`bounceOutLeft`、`bounceOutRight`、`bounceOutUp`|fadeOut|

代码结构及示例：
```
[mask_off time="1000" effect="fadeOut"]
```
